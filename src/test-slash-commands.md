# 斜杠命令功能测试指南

## 如何测试斜杠命令功能

1. 打开浏览器访问 http://localhost:5174/
2. 在编辑器中点击任意位置获得焦点
3. 输入 `/` 字符
4. 应该会看到一个命令面板弹出，显示可用的命令选项
5. 使用上下箭头键导航选项
6. 按 Enter 键或点击选项来插入对应的内容

## 可用的命令

- **标题 1** (H1) - 创建大标题
- **标题 2** (H2) - 创建中等标题  
- **标题 3** (H3) - 创建小标题
- **无序列表** (•) - 创建项目符号列表
- **有序列表** (1.) - 创建编号列表
- **引用** (") - 创建引用块
- **代码块** (</>) - 创建代码块
- **分割线** (—) - 插入水平分割线

## 功能特性

- ✅ 输入 `/` 触发命令面板
- ✅ 支持键盘导航（上下箭头键）
- ✅ 支持鼠标点击选择
- ✅ 支持搜索过滤（输入文字过滤命令）
- ✅ 按 Escape 键关闭面板
- ✅ 美观的 UI 设计
- ✅ 响应式布局

## 测试场景

1. **基本功能测试**
   - 输入 `/` 查看面板是否出现
   - 选择不同的命令验证是否正确插入内容

2. **搜索功能测试**
   - 输入 `/标题` 查看是否只显示标题相关选项
   - 输入 `/列表` 查看是否只显示列表相关选项

3. **键盘导航测试**
   - 使用上下箭头键导航
   - 按 Enter 键选择
   - 按 Escape 键取消

4. **边界情况测试**
   - 在不同位置输入 `/`
   - 在已有内容中间输入 `/`
   - 连续输入多个 `/`
